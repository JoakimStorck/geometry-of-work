# O*NET Database 30.1 (subset)

This directory contains a subset of tables from the O*NET 30.1
Database, December 2025 release, unmodified. Only the tables used by
this project's pipeline are included: Task Statements, Task Ratings,
Task Categories, Occupation Data, Skills, Abilities, and Education,
Training, and Experience. The Job Families table and the SOC
crosswalk under `data/onet/` come from the same source.
